# Generated README Assets

These bitmap assets were generated with the built-in `imagegen` tool for the
developer-facing README refresh. Keep them in the repo because the README
references project-local paths.

| File | Purpose | Prompt intent |
|---|---|---|
| `goal-banner.png` | README banner | Premium developer-tool banner for `goal`, showing durable objectives, a statusline cockpit, and an audit trail. |
| `diagrams/goal-lifecycle.png` | Core loop diagram | Objective -> `goalframe` -> `.goal` state -> continuation -> `overclaim` -> achieved. |
| `diagrams/companion-skills.png` | Skills diagram | `goalframe` at intake and `overclaim` at exit around a durable goal record. |
| `diagrams/cowork-relay.png` | Cowork relay diagram | Claude Code and Codex as interchangeable runners against one shared `.goal` state directory. |

Constraints used for the prompts: no personal names, no local machine paths, no
API keys, no vendor logos, no watermarks, and no critical facts available only
inside the image.
